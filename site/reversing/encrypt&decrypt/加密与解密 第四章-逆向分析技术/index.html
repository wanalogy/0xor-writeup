```c /********* 《加密与解密》第三版配套实例 4.4 虚函数 (c) 看雪软件安全网站 www.pediy.com 2000-2008 ********/

#include class CSum { public: virtual int Add(int a, int b) { return (a + b); }

{
    return (a - b);
}

};

void main() {

pCSum->Sub(1,2);

}

```
